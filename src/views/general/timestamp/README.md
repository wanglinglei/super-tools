# 时间戳转换工具

在线时间戳与日期时间互相转换工具，支持秒和毫秒格式，提供快捷预设功能。

## 功能特性

### 1. 当前时间显示
- ✅ 实时显示当前时间戳（秒）
- ✅ 实时显示当前时间戳（毫秒）
- ✅ 显示格式化的日期时间
- ✅ 每秒自动更新
- ✅ 一键复制时间戳

### 2. 时间戳 → 日期时间
- ✅ 输入任意时间戳
- ✅ 自动识别秒/毫秒格式
- ✅ 手动选择时间单位
- ✅ 显示本地化日期时间格式
- ✅ 显示 ISO 8601 格式
- ✅ 一键复制结果

### 3. 日期时间 → 时间戳
- ✅ 日期时间选择器
- ✅ 同时显示秒和毫秒格式
- ✅ 实时转换
- ✅ 一键复制结果

### 4. 快捷预设
提供 8 个常用时间快捷按钮：
- 今天 00:00
- 今天 23:59
- 昨天此时
- 明天此时
- 一周前
- 一周后
- 一个月前
- 一个月后

## 使用说明

### 查看当前时间戳
1. 打开页面即可看到当前时间戳
2. 每秒自动更新
3. 点击复制按钮快速复制

### 时间戳转日期
1. 在"时间戳 → 日期时间"区域输入时间戳
2. 选择单位（自动识别/秒/毫秒）
3. 自动显示转换结果
4. 点击复制按钮复制结果

### 日期转时间戳
1. 在"日期时间 → 时间戳"区域选择日期时间
2. 自动显示秒和毫秒两种格式
3. 点击对应复制按钮复制结果

### 使用快捷预设
1. 点击快捷按钮（如"今天 00:00"）
2. 自动填充日期时间并转换为时间戳
3. 快速获取特定时间的时间戳

## 技术实现

### 时间戳识别
```typescript
// 自动识别单位
if (timestampUnit.value === 'auto') {
  // 10位数字认为是秒，13位数字认为是毫秒
  if (timestamp.toString().length === 10) {
    timestamp = timestamp * 1000;
  }
}
```

### 实时更新
```typescript
// 每秒更新当前时间
timeUpdateTimer = setInterval(() => {
  currentTime.value = Date.now();
}, 1000);
```

### 日期格式化
```typescript
// 本地化格式
date.toLocaleString('zh-CN', {
  year: 'numeric',
  month: '2-digit',
  day: '2-digit',
  hour: '2-digit',
  minute: '2-digit',
  second: '2-digit',
  hour12: false,
});

// ISO 8601 格式
date.toISOString();
```

## 支持的格式

### 输入格式
- Unix 时间戳（秒）：10 位数字
- Unix 时间戳（毫秒）：13 位数字
- 日期时间选择器标准格式

### 输出格式
- 本地化格式：`2026/01/28 15:30:45`
- ISO 8601 格式：`2026-01-28T07:30:45.000Z`
- Unix 秒：`1737984645`
- Unix 毫秒：`1737984645000`

## 注意事项

1. 时间戳范围建议在合理范围内
2. 自动识别基于数字位数（10位/13位）
3. 日期时间选择器使用本地时区
4. 复制功能需要浏览器支持 Clipboard API
5. 时间显示使用中国标准时间（CST）
